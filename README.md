# Demo Add-in para Excel (Office.js)

Este repositório contém um exemplo simples de add-in para Microsoft 365 Excel usando Office.js.

Como usar:
1. Altere o manifest.xml com as URLs corretas do seu host.
2. Publique o manifest no catálogo de suplementos da sua organização ou carregue localmente no Office (desenvolvimento).
3. Carregue o add-in no Excel via Inserir > Meus Suplementos > Carregar Complemento (manifest.xml).

Observações:
- Este exemplo funciona para demonstração e aprendizado.
- Em produção, implemente tratamento de erros, autenticação e segurança de origem.
